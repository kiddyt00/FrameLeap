# 阶段5：分镜编排阶段详细说明

## 1. 概述

分镜编排阶段负责将生成的图像按照剧本和视觉叙事规则组织成完整的镜头序列。本阶段确定每个镜头的时长、顺序、转场方式，确保整体节奏和叙事流畅性。

---

## 2. 镜头序列规划

### 2.1 时间轴编排

#### 镜头时长计算

**基于内容的时长计算**：
- 对白镜头 = 字数 × 阅读速度(0.3秒/字) + 缓冲时间(0.5-1秒)
- 动作镜头 = 动作完整展示时长 + 进入/退出缓冲
- 情感镜头 = 基础时长(2秒) + 情感强度系数 × 延长时间
- 环境镜头 = 场景复杂度 × 0.5秒 + 最小停留(1.5秒)

**时长类型规划**：

| 镜头类型 | 时长范围 | 说明 | 节奏作用 |
|---------|---------|------|---------|
| 快速反应 | 0.5-1秒 | 惊讶、快速切换 | 制造冲击、紧张 |
| 短镜头 | 1-2秒 | 动作场景、快速剪辑 | 推进节奏、动感 |
| 标准镜头 | 3-5秒 | 大多数对话和场景 | 舒适观看、信息传递 |
| 长镜头 | 5-8秒 | 情感表达、环境展示 | 情感沉淀、氛围营造 |
| 超长镜头 | 8秒以上 | 特殊强调、慢镜头 | 深度沉浸、仪式感 |

**时间轴数据结构**：

### 2.2 镜头类型分配

#### 建立镜头（Establishing Shot）

**用途与特征**：
- 功能：交代场景环境、定位时空关系
- 景别：远景/大远景
- 时长：2-5秒
- 运镜：缓慢摇摄或固定

**生成规则**：
- 场景首次出现时必须使用
- 包含环境标志性元素
- 若有角色，角色占比不超过30%
- 使用黄金分割构图
- 光线体现场景氛围

**叙事时机**：
| 时机 | 说明 | 示例 |
|-----|------|------|
| 故事开场 | 建立世界观 | "赛博都市夜景，霓虹闪烁" |
| 场景转换 | 切换到新地点 | "从室外进入室内" |
| 时间过渡 | 展现时间流逝 | "从黄昏到夜晚的延时" |

#### 对话镜头（Dialogue Shot）

**对话镜头编排模式**：
| 模式 | 描述 | 镜头安排 | 适用场景 |
|-----|------|---------|---------|
| 单人镜头 | 每个角色单独拍摄 | A说话→A镜头，B说话→B镜头 | 独白、强烈情感 |
| 过肩镜头 | 经典对话方式 | A说话→越过B看A | 标准对话、关系展示 |
| 双人镜头 | 两人同框 | 中景展示两人互动 | 亲密关系、平等对话 |
| 反应镜头 | 展示听者反应 | A说话→B的反应特写 | 冲突、秘密、情绪戏 |

**对话镜头生成逻辑**：
```
IF 对话角色数 == 2:
    IF 情感强度 > 高:
        推荐模式 = [单人镜头交替 + 反应镜头]
    ELSE IF 关系亲密:
        推荐模式 = 双人镜头
    ELSE:
        推荐模式 = 过肩镜头
```

#### 情绪特写（Emotional Close-up）

**特写使用规则**：
- 角色情感达到峰值时使用
- 每个情感高潮最多1-2个特写
- 特写前应有中景/近景铺垫
- 特写后应有释放镜头

**特写类型**：
| 类型 | 聚焦区域 | 叙事目的 | 时长建议 |
|-----|---------|---------|---------|
| 面部特写 | 面部表情 | 展现情感 | 2-4秒 |
| 眼睛特写 | 眼睛细节 | 传递内心活动 | 1-3秒 |
| 嘴部特写 | 嘴部微表情 | 细腻情感变化 | 1-2秒 |
| 手部特写 | 手部动作/姿态 | 紧张、决断、亲密 | 2-3秒 |
| 物品特写 | 关键物品 | 象征、线索 | 1.5-3秒 |

#### 动作镜头（Action Shot）

**动作镜头编排**：
| 动作类型 | 镜头策略 | 景别 | 时长 |
|---------|---------|------|------|
| 准备动作 | 展示蓄力过程 | 全景/中景 | 中等 |
| 动作爆发 | 强调速度和力量 | 近景/特写 | 短 |
| 动作结果 | 展示影响和后果 | 中景/全景 | 中长 |
| 连续动作 | 多角度组合 | 混合 | 多短镜头 |

### 2.3 镜头顺序优化

#### 视觉连贯性检查

**检查项**：
- 方向一致性（180度规则）
- 视角变化幅度（≥30度原则）
- 视觉焦点连续性
- 色彩/光线平滑过渡

#### 自动镜头重排

**重排规则**：
```
IF 相邻镜头角度差 < 30度:
    调整运镜或插入中间镜头
IF 角色视线不连贯:
    添加反应镜头或视线引导镜头
IF 色彩突变:
    添加过渡色彩或调整调色
```

## 3. 节奏控制

### 3.1 整体节奏规划

#### 节奏曲线生成

**节奏类型映射**：
| 故事阶段 | 节奏特征 | 平均镜头时长 | 剪辑频率 | 运镜强度 |
|---------|---------|-------------|---------|---------|
| 铺垫期 | 平缓、渐进 | 4-6秒 | 低 | 弱 |
| 发展期 | 波动上升 | 3-5秒 | 中 | 中 |
| 冲突期 | 快速、紧张 | 1-3秒 | 高 | 强 |
| 高潮期 | 密集、爆发 | 0.5-2秒 | 极高 | 极强 |
| 结局期 | 缓慢、释放 | 5-8秒 | 低 | 弱 |

**节奏曲线示例**：
```
紧张度
  ↑
高│           ╱╲
  │         ╱  ╲
  │       ╱      ╲___
中│     ╱╲          ╱
  │   ╱  ╲        ╱
  │ ╱     ╲______╱
低│╱
  └────────────────────→ 时间
   开场  发展  高潮  结局
```

### 3.2 段落节奏调整

#### 快节奏段落

**特征与技巧**：
| 技巧 | 实现方式 | 效果 |
|-----|---------|------|
| 短镜头快速切换 | 1-2秒镜头，频繁切 | 紧张感、冲击力 |
| 镜头类型变化 | 远→中→近快速推进 | 加速感 |
| 运镜加速 | 缩放、甩镜头 | 动感增强 |
| 音效配合 | 密集音效、节奏音乐 | 声画共振 |
| 特效叠加 | 速度线、冲击波 | 视觉冲击 |

**适用场景**：
- 冲突对抗
- 追逐逃跑
- 危机时刻
- 情绪爆发

#### 慢节奏段落

**特征与技巧**：
| 技巧 | 实现方式 | 效果 |
|-----|---------|------|
| 长镜头停留 | 5-8秒甚至更长 | 沉淀情感、细节观察 |
| 缓慢运镜 | 超慢推拉摇移 | 浸入感、凝视感 |
| 固定构图 | 静态画面 | 稳定、安宁 |
| 柔和转场 | 长淡入淡出 | 平滑过渡 |
| 环境音铺底 | 自然环境音 | 氛围沉浸 |

**适用场景**：
- 抒情独白
- 环境展示
- 情感沉淀
- 回忆梦境

### 3.3 高潮处理

#### 高潮镜头密度

**高潮层次设计**：
| 层次 | 功能 | 镜头密度 | 时长占比 |
|-----|------|---------|---------|
| 高潮前奏 | 积蓄张力 | 中高 | 15% |
| 高潮上升 | 推进冲突 | 高 | 30% |
| 高潮顶点 | 爆发点 | 极高 | 10% |
| 高潮回落 | 释放 | 中 | 25% |
| 余韵 | 延展 | 低 | 20% |

**高潮节奏示意图**：
```
镜头密度（镜头数/分钟）
  ↑
高│            ／＼
  │          ／    ＼
  │        ／        ＼
中│      ／            ＼____
  │    ／                  ＼
  │  ／                      ＼
低│＿＿                        ＼＿
  └─────────────────────────────→ 时间
   前奏  上升  顶点  回落  余韵
```

**高潮镜头编排原则**：
1. 逐层推进：景别由大到小，节奏由慢到快
2. 多角度呈现：同一动作多角度快速切换
3. 特效叠加：在顶点处叠加最强视觉效果
4. 声画共振：音效、音乐与画面同步达到峰值
5. 及时释放：顶点后迅速放缓，避免审美疲劳
