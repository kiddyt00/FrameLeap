# 阶段5：分镜编排阶段详细说明

## 1. 概述

分镜编排阶段负责将生成的图像按照剧本和视觉叙事规则组织成完整的镜头序列。本阶段确定每个镜头的时长、顺序、转场方式，确保整体节奏和叙事流畅性。

## 4. 转场设计

### 4.1 转场类型

#### 基础转场

| 转场类型 | 持续时间 | 使用场景 | 效果 |
|---------|---------|---------|------|
| 切换（Cut） | 0秒 | 动作连贯、快速剪辑 | 直接、干脆 |
| 淡入淡出（Fade） | 0.5-2秒 | 场景转换、时间流逝 | 柔和、过渡 |
| 擦除（Wipe） | 0.5-1秒 | 平行叙事、场景切换 | 清晰分隔 |
| 溶解（Dissolve） | 1-2秒 | 回忆、梦境 | 融合、模糊 |
